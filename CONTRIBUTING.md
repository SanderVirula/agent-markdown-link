# Contributing

Thanks for helping. Please open an issue before proposing a larger change, keep changes scoped, and preserve the local-only, review-first model.

Use Node.js 22 or newer, run `npm ci`, then run the relevant checks before opening a pull request. Do not add telemetry, automatic network activity, automatic Git or Obsidian actions, or canonical-note mutation.
